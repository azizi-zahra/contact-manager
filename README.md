# Contact Manager

A contact management web application built with FastAPI to practice backend development concepts, REST API design, and CRUD operations.

## About

This project is a backend-focused application for managing contacts. It was created as a learning project to explore FastAPI and strengthen backend development skills through practical implementation.

## Features

* Create new contacts
* View saved contacts
* Update contact information
* Delete contacts
* RESTful API structure

## Purpose

This project is part of my learning process in backend and web development, with a focus on building practical applications and improving API design skills.
